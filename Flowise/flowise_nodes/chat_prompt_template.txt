=> Chat Prompt Template

    -> Purpose

        -> Creates the final prompt that will be sent to the LLM.
        -> Combines the System Message and Human Message into a single prompt.
        -> It does not communicate with the LLM directly.

    -> Components

        -> System Message

            -> Defines the behavior or personality of the AI.

            -> Examples

                -> You are a Java Expert.
                -> Answer only in JSON.
                -> You are a Travel Assistant.

        -> Human Message

            -> Represents the user's actual input.

            -> Examples

                -> What is Spring Boot?
                -> Explain Docker Volumes.

    -> Flow

            System Message
                    +
            Human Message
                    ↓
            Final Prompt
                    ↓
            Send to LLM Chain

    -> Important

        -> The Prompt Template only prepares the prompt.
        -> It never communicates directly with the LLM.
